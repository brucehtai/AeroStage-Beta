# Changelog

## v0.2.0-beta

- Rebuilt AeroStage Beta as a six-level public beta.
- New public level names:
  - 0 Reference / Bypass
  - 1 Comfort
  - 2 Diffusion
  - 3 Bloom
  - 4 Atmosphere
  - 5 Concert
- Updated DSP mapping to selected AeroStage levels 0, 2, 4, 6, 8, and 10.

- Removed misleading fake controls; the GUI shows only the real preset selector.
- No extra post-output trim added beyond the tuned preset headroom.
